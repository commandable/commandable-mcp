# Commandable MCP — Static Mode

You are connected to **Commandable** in static (compatibility) mode. All configured tools are loaded up front.

## What Commandable gives you

Pre-built tools for popular services (GitHub, Notion, Trello, Jira, Google Workspace, HubSpot, and more) — all available immediately, as well as custom tools that may have already been configured in create mode. 

## How it works

- Every tool from every configured integration is already registered in this session.
- Just call the tools you need — no search or enable step required.
- This mode is designed for clients that do not support MCP dynamic tool loading.

## Other features 

Commandable also provides a create mode - where connected agents can add new integrations and connections in place. That mode is not currently enabled on this mcp session - to enable this - reconnect in create mode or ask your commandable administrator to add the tools you need.
