Obtain a short-lived Google OAuth access token using these scopes:

- `https://www.googleapis.com/auth/drive.file`
- `https://www.googleapis.com/auth/documents`
- `https://www.googleapis.com/auth/spreadsheets`
- `https://www.googleapis.com/auth/presentations`

You can use the Google OAuth 2.0 Playground (`https://developers.google.com/oauthplayground/`) or your own OAuth flow.

Note: OAuth access tokens are short-lived (typically 1 hour). For long-running use or full enterprise access, prefer the Service Account variant.
